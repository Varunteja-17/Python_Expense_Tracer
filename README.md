# Python Expense Tracker using MySQL

A menu-driven **Expense Tracker application** developed using **Python and MySQL** that allows users to record, store, and analyze daily expenses.  
The project focuses on **database-driven storage**, eliminating the use of text files and providing a structured, scalable solution suitable for real-world applications.
